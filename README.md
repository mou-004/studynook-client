StudyNook – Library Study Room Booking

A full-stack MERN web application where students and library users can discover, book, and manage private study rooms with secure authentication and real-time booking conflict prevention.

🌐 Live Website

🔗 Client Live URL:https://studynook-client-2-7ccc.onrender.com

📖 Project Overview

StudyNook is a responsive and recruiter-friendly study room booking platform built with the MERN stack. Users can browse available study rooms, filter and search rooms, and book available time slots without conflicts. Room owners can manage their listings while users can monitor and cancel bookings from their personalized dashboard.

The platform uses JWT authentication stored in HTTP-only cookies to ensure secure access to private routes and sensitive operations.

✨ Key Features

🔐 Secure JWT Authentication with HTTP-only cookies

🏠 Add, Update, and Delete Study Room Listings

📅 Real-Time Booking System with Time Conflict Detection

🔍 Search and Filter Rooms by Amenities and Room Name

📱 Fully Responsive Design for Mobile, Tablet, and Desktop

⚡ Dynamic Page Titles and Smooth UI Animations

📊 User Dashboard for Managing Listings and Bookings

🚫 Prevents Double Booking for the Same Time Slot

🔔 Toast Notifications for Success and Error Handling
